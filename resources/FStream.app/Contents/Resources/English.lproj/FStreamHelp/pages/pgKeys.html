Modifier keys

Modifier keys allows you to modify the default FStream behavior, as described below.

  • Play panel (no key):
    FStream display is in standard mode (audio stream playing).
    • The info area shows the information described int the Interface section.
    • The controller can show:
      • A “Play” symbol (arrow), meaning that either there is no loaded stream, or that the current stream is paused. Clicking on it either will load the selected web radio (from the Streams List), or will resume play of the paused stream.
      • A “Pause” symbol (double bar), meaning that a web radio stream is loaded and playing. Clicking on it will pause playback of the stream (see the different pause modes in the Preferences section).
      • A wave symbol means that FStream is loading a radio stream.
    • The main window shows a yellow background.
  • Record panel (Shift key):
    Accesses FStream’s record mode.
    • The info area shows the length of the current recording session in hhh:mm:ss format.
    • The controller will show either:
      • A “Record” symbol (circle), meaning either that there is no current recording session, or that the recording session is paused. Clicking on it either will start a new recording session (creating a new file at the location set in Preferences > “Record” tab > “Record folder” button) and reset the counter, or will resume the paused recording session.
      • A “Pause” symbol (double bar), meaning that FStream is recording the audio output. Clicking on it will pause the recording session.
    • The main window shows a light-red background.
  • Stop play/record (Option key):
    • If you are in play mode (Shift key not pressed), and a stream is loaded, the controller will show a “Stop” symbol (square). Clicking on it will stop the loaded stream.
    • If you are in record mode (Shift key pressed), and a recording session is in progress, the controller will show a “Stop” symbol (square). Clicking on it will just stop the recording session and close the recorded audio file.
  • Reload current stream ("r" key):
    If an audio stream is loading or is loaded, the controller will show a “Reloading” symbol (3/4 of a circle with an arrow). Clicking on it will reload the stream.
  • Simulate controller click (Space bar):
    To simulate a click on the controller, just press the space bar.
