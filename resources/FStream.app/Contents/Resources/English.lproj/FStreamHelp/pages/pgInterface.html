Interface presentation

Because of willingness to do a ultra-condensed interface, the role of the interface elements should be explained.

  • The Controller (on the left):
    Allows you to control FStream. The use of modifier keys allows you to switch between playing and recording modes.
  • The Info Area (in the middle):
    Allows you to show the record time, or the following five items of information, which change every five seconds (if turned on in preferences):
    1. percentage of the buffer thatâ€™s filled
    2. stream status
    3. song artist*
    4. song title*
    5. song album*
    * displayed only if the radio station provides this information in its stream
  • The Contextual Menus (on the right):
    Clicking on one of the two half-circles accesses a contextual menu.
    • The upper half-circle provides access to the Streams Manager, Quick Open, Preferences, and the Streams List.
    • The lower half-circle provides access to a volume slider.
