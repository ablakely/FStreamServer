Préférences

FStream est un petit logiciel, mais il y a de nombreuses choses personnalisables. Vous pouvez trouver ici une description des préférences.

  • Divers:
    • Télécommande Apple IR Remote: vous permez d'activer ou désactiver la télécommande dans FStream (de manière exclusive ou non).
    • Fenêtre principale: permet de spécifier si la fenêtre principale de FStream soit se trouver par dessus les autres ou non.
    • Menu statut: permet d'afficher un menu de statut à droite de votre barre de menu pour contrôler FStream de n'importe où.
    • Mise à jour: activez cette option pour permettre à FStream de vérifier automatiquement la présence de mise à jour à chaque lancement du logiciel. Le bouton "Maintenant" permet d'effectuer cette vérification de suite.
  • Apparence:
    Permet de changer l'apparence de la fenêtre principale de FStream. Référez-vous au site SourceMac.com pour en télécharger et savoir comment en fabriquer.
    • Pour changer d'apparence, sélectionnez l'apparence que vous voulez utiliser dans la liste. Les informations sur l'apparence sélectionné apparaissent dans le cadre de droite.
  • Info:
    Permet de configurer les informations affichés dans la fenêtre principale de FStream.
    • Changement d'information permet de dire si FStream doit automatiquement changer l'information affiché toutes les 5 secondes (activé ou non, un click sur la zone d'information permet de toute façon de changer l'information affichée) et d'activer/désactiver l'animation de défilement.
    • Information WebRadio permet de choisir l'encodage d'affichage des informations des WebRadio, de ne pas prendre en compte une information si elle contient un élément définit (exemple "À suivre :") et de changer la casse des informations.
    • iChat permet de changer automatiquement le statut d'iChat avec le titre de la musique et le nom de l'artiste quand ils changent.
  • Événements:
    Permet d'attraper des événements de FStream. Les événements peuvent être envoyés vers l'historique (vous pouvez accéder à la fenêtre d'historique via le menu Fenêtre) et/ou vers Growl.
  • Enregistrement:
    Permet de configurer l'enregistrement des streams.
    • Dossier d'enregistrement: permet de choisir le dossier de destination pour les fichiers audio.
    • Format permet de choisir le format des fichier audio (mp3/aac/aiff/wav).
    • Qualité permet de spécifier le débit (en kbps) et la qualité de l'algorithme d'encodage pour le format d'enregistrement mp3. Si vous voulez utiliser le même débit que la web radio pour votre enregistrement, utilisez "Auto" dans le menu débit.
  • Pause:
    Permet de définir le comportement de la pause dans le panneau de lecture.
    • Pause (silence uniquement): dans ce mode, un click sur le bouton pause du contrôleur va simplement couper la sortie audio et couper l'enregistrement si une session est en cours. En sortant de pause, vous allez simplement réentendre le flux tel qu'il est actuellement diffusé.
    • Pause (silence, mise en cache pour reprise): dans ce mode, un click sur le bouton pause du contrôleur va alors enregistrer le flux audio dans le dossier de cache (que vous pouvez changer avec le bouton "Dossier de cache") et couper l'enregistrement si une session est en cours. En sortant de pause, vous allez alors simplement reprendre la lecture du flux audio là où vous l'avez mis en pause, via l'utilisation des fichiers de cache. L'écoute n'est alors bien sûr plus en temps réel par rapport au flux audio tel qu'il est diffusé.
      La cache est un groupe de fichier, chacun étant coupé à une taille maximum de 100 Ko. Quand un fichier n'est plus utilisé, il est supprimé.
  • Parole:
    Permet de configurer la synthèse vocale dans FStream.
    • Éléments prononcés
      Activer ces options a pour effet de faire prononcer par la synthèse vocale d'Apple le nom de la radio, et/ou prononcer le nom, l'artiste et l'album de la musique quand ils changent. (La voix utilisée pour la prononciation peut être changée dans le panneau Parole des Préférences Système [Préférences Système > Parole > Synthèse vocale: Voix système].)
    • Activation de la parole
      'Circonstance' permet de configurer quand la parole est déclenchée : au chargement de la web radio, ou après.
      'Télécommande' permet d'activer la parole seulement quand vous contrôlez FStream via la télécommande d'Apple.
  • Connexion:
    Permet de configurer la connexion Internet de FStream.
    • Cache:
      La case à cocher permet de définir le pourcentage de remplissage à partir duquel FStream doit recharger le stream. Utile si le stream n'est plus reçu pendant un certain laps de temps (déconnexion par exemple).
      Le pop-up menu "Mémoire cache" permet de définir la taille de la cache (Petite pour une connexion ADSL stable, Grande pour une connexion 56k assez stable).
    • Reconnexion:
      Permet de dire à FStream de se reconnecter un certain nombre de fois, tant que la connexion échoue.
    • Proxy:
      Vous avez la possibilité d'utiliser un proxy SOCKS (version 4 ou 5) pour toutes les connexions Internet de FStream (par exemple si vous vous trouvez dans un réseau nécessitant un accès proxy pour accéder à Internet).
